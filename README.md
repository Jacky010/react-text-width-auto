> 文本宽度自适应组件

## Install

```sh
npm install
```

## Start

```sh
npm run dev
```

## Run Build

```sh
npm run build
```

## Use

```sh
import WidthAutoLabel from 'react-text-width-auto/src/index';
```

可通过设置外部容器的`color`来设置自适应后的文本颜色，如下：
```
<div 
    style={{
        width: 100,
        height: 32,
        fontSize: '14px',
        fontWeight: 'bold',
        border: '1px solid #f00',
        color: red
    }}
>
    <WidthAutoLabel>测试文本宽度自适应标签组件</WidthAutoLabel>
</div>
```
